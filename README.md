This project is a responsive Amazon clone website built using HTML and CSS. 
It replicates the basic layout and design of the Amazon homepage, including the navigation bar, search bar, product sections, and footer. 
The main focus of this project is to practice front-end development skills such as layout design, Flexbox, and responsive web design using media queries.
The website adjusts smoothly across different screen sizes like mobile, tablet, and desktop.
This project helped in understanding modern UI structure and improving styling techniques.
It is a beginner-friendly project created for learning and improving web development fundamentals.
